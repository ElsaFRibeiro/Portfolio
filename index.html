XtremeMac Website Redesign

XtremeMac paved their way when they launched the very first case for the iPod in 2001. Since then, they have been driven by passion to provide the top innovative accessories for apple and android devices.

Task

The task at hand was to redesign the information architecture, layout, and user flow of the website. It was also necessary to update the website according to the new business strategy and brand style guide.

Process

It was essencial to understand what the users common behaviours were withing the website therefore I created user strories based on data-driven marketing personas. I selected a couple of websites that our customer base engages with and tried to suggest improvements on the flow and user experience. The websiteÂ´s responsiveness was also created, it did not previously adapt to different screen resolutions nor appear optimally in a range of devices. Imgaes, Typography and Ensuring an easy navigation was also improved in order to Boost SEO gains.

Low fidelity & High Fidelity Wireframes

Once we understood the user stories and the necessary funcionality of the website I passed on to creating the low fidelity wireframes and then moved on to high fidelity mocks using Adobe XD.

Implementation

This website was developed using WordPress as a CMS, Elementor Plugin, HTML and CSS.

Result

365 Alert Website Creation

A website that allows people to subscribe and receive alerts of possible real-time floods in Luxembourg.

Task

Creation of a website that sends an alert notification to the user according to the possibility of a flooding within 13 different locations in Luxembourg. The website receives the information on the level of water from an API, updated every 15 minuts. There are 3 different type of alerts, each one has its own specific information on the level of danger and associated safety procedures to follow.

Process

Creation of user personas in order to understand their common behaviours as well as taking data already collected by similar applications.

Low fidelity & High Fidelity Wireframes

Once we understood the user stories and the necessary funcionality I passed on to creating the low fidelity wireframes and then moved on to high fidelity mocks using Adobe XD.

Implementation and Testing

This website was then developed using the Laravel Framework, HTML, CSS, Javascript and PHP languages and ultimately tested by various team members according to the user personas.
